# Shell Redirections Project

This project contains basic shell scripts that demonstrate input/output redirections and filters in Bash.

Each script performs a specific task using commands like:
- cat
- head
- tail
- find
- wc
- sort
- uniq
- grep
- tr
- cut
- rev

All scripts follow Holberton School requirements:
- Exactly 2 lines
- Start with #!/bin/bash
- Executable files
